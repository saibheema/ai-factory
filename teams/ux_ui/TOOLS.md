# UX / UI Design Team — TOOLS

Research:
- **Firecrawl**: Crawl competitor/reference product UIs for UX pattern research
- **Playwright**: Accessibility audit on reference sites (WCAG 2.1 AA checks)

Documentation:
- **Google Docs**: UX Flow Specification (user journeys, screen inventory, design tokens, component map)
- **Mermaid**: User flow diagrams (Flowchart LR — all navigation paths)

Tracking:
- **Plane**: UX issues linked to PRD stories, assigned to Frontend Eng

Notifications:
- **Slack**: #design channel — UX spec published with Doc link + design token summary
- **Notification**: Stage-complete broadcast to Frontend Eng
