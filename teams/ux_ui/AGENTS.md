# UX / UI Design Team — AGENTS

Operatives:
- **Lead UX Designer** — owns user flows, persona journeys, information architecture
- **UI Specification Author** — documents every screen with component inventory and state matrix
- **Design Tokens Keeper** — defines and maintains color, typography, spacing, radius tokens
- **Component Mapper** — maps each UI element to the component library chosen by Sol Arch
- **Accessibility Auditor** — runs Playwright accessibility checks, ensures WCAG 2.1 AA compliance
- **Competitive Researcher** — uses Firecrawl to crawl reference products and extract UX patterns
- **Interaction Designer** — specifies hover, focus, transition, animation, and gesture behaviors

Handoff Protocol:
  Lead UX Designer publishes UX Flow Specification in Google Docs.
  Design token JSON is included inline (copy-paste ready for Frontend Eng).
  Playwright a11y report is attached.
  Frontend Eng receives Slack notification before wave 4 starts.
