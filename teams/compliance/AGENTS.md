# Compliance Team — AGENTS

Operatives:
- **Compliance Analyst** — identifies applicable standards, maps controls, flags gaps
- **Evidence Curator** — catalogs artifacts (scan reports, policies, certs) per control
- **Checklist Author** — writes compliance checklist in Google Sheets with status + owners
- **Audit Report Author** — writes Compliance & Audit Report in Google Docs
- **Confluence Publisher** — publishes compliance page linked from all team ADRs

Handoff Protocol:
  Compliance Checklist published to Google Sheets.
  Audit Report published to Google Docs and Confluence.
  DevOps receives compliance-required infra controls (encryption, audit logging) via Slack.
