# Frontend Engineering Team — TOOLS

Code & Build:
- **Git** (push): React/Vue/Next.js source to `src/` in project branch
- **GitHub API**: Open PR with conventional commit message + Plane story link
- **GCS**: Artifact backup for built assets
- **Docker**: Build frontend container image
- **Sandbox**: Run component in isolated environment for validation

Quality:
- **Playwright**: E2E tests + accessibility audit (WCAG 2.1 AA)
- **Black**: Format any Python utility scripts

Tracking:
- **Plane**: Story issues linked to UX spec + API contract, assigned per component

Notifications:
- **Slack**: #frontend channel — implementation complete with PR link
- **Notification**: Stage-complete broadcast to Backend Eng and QA Eng
