# Feature Engineering Team — TOOLS

Tracking:
- **Google Sheets**: Feature Tracker (Feature ID, story, priority, status, sprint, owner, points)
- **Plane**: Close done issues, defer blocked ones, update sprint milestone
- **Jira**: Mirror Plane status updates to Jira for cross-org visibility

Documentation:
- **Google Docs**: Feature Closure Report (features delivered, velocity, quality gate status)

Notifications:
- **Slack**: #delivery channel — sprint closure summary with velocity metrics
- **Notification**: Stage-complete broadcast to Product Mgmt for next sprint kickoff
