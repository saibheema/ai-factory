# Feature Engineering Team — AGENTS

Operatives:
- **Feature Tracker Lead** — maps features to stories, verifies acceptance criteria closure
- **Sprint Accountant** — calculates velocity, cycle time, quality rate per sprint
- **Backlog Curator** — closes done issues in Plane + Jira, defers blocked items to next sprint
- **Feature Closure Author** — writes the Feature Closure Report (Google Docs)
- **Backlog Prioritization Analyst** — re-ranks remaining backlog for next iteration

Handoff Protocol:
  Feature Tracker spreadsheet updated (Google Sheets).
  Feature Closure Report published (Google Docs).
  Plane + Jira issues all closed or re-scheduled.
  Product Mgmt receives velocity + quality metrics via Slack for next sprint planning.
