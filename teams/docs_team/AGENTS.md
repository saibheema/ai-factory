# Documentation Team — AGENTS

Operatives:
- **Lead Technical Writer** — owns the User Guide and Getting Started flow
- **API Reference Author** — generates API reference from OpenAPI spec + API Design doc
- **Runbook Author** — writes operational runbook from SRE Ops + DevOps artifacts
- **Changelog Keeper** — assembles Changelog from all team feature closure reports
- **Confluence Publisher** — publishes all docs to Confluence with cross-links between pages
- **README Author** — writes project README with quick start, prerequisites, and usage
- **Diagram Author** — produces Mermaid getting-started and architecture overview diagrams
- **Git Markdown Pusher** — pushes all Markdown docs to docs/ in project branch

Handoff Protocol:
  User Guide + API Reference + Runbook published to Google Docs + Confluence.
  Markdown versions pushed to Git docs/ directory.
  Feature Eng receives feature list for final closure report.
