# Business Analysis Team — TOOLS

Research:
- **Tavily Search**: domain research, regulatory requirements, competitor feature analysis
- **Firecrawl**: crawl reference documentation and industry standards
- **Wikipedia**: domain context and terminology

Documentation:
- **Google Docs**: Business Requirements Document (BRD — primary deliverable)
- **Google Sheets**: Acceptance Criteria Matrix (story ID, criterion, priority, status, owner)
- **Confluence**: Published BRD page linked from Sol Arch ADR
- **CSV**: Import/export acceptance criteria for external stakeholder review

Tracking:
- **Plane**: Story issues per acceptance criterion, linked to PM Epics
- **Jira**: Mirror stories for cross-org tracking

Notifications:
- **Slack**: #biz-analysis channel — BRD published with acceptance criteria count
- **Notification**: Stage-complete broadcast to Sol Arch
